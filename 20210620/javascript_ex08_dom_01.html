요소(element)를 선택하는 방법

  • 선택자로 선택하는 방법
    1. 요소객체.querySelector("선택자")
      • 하나의 요소를 선택하는 방법
      • 가장 위쪽아이만 선택 된다
    2. 요소객체.querySelectorAll("선택자")
        복수로 선택하는 방법
  • 태그로 선택하는 방법
    1. 태그로 선택 - document.getElementsByTagName("태그명")
      • 배열로 선택한 객체를 가져온다.
  • 속성으로 선택하는 방법
    1. id로 선택 - document.getElementById("id명")
      • 하나의 객체만 선택된다
    2. class로 선택-document.getElementsByClassName("클래스명")
      • 배열의 형태로 선택됨
  • 요소간의 관계로 선택하는 방법
    1. 노드(텍스트노드, 엘리먼트노드)로 접근한다.
    2. 부모
      • 요소객체.parentNode
    3. 형제
      • 다음형제-요소객체.nextSibling
      • 이전형제-요소객체.previousSibling
    4. 자식
      • 자식들-요소객체.childNodes
      • 마지막자식-요소객체.lastChild
      • 자식중 element만 선택 - 요소객체.children
      • 마지막 element 자식 선택 - 요소객체.lastElementChild
